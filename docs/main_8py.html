Detailed Description

Task implementation for Interrupt Service Routine RC circuit step Response.

Creates an Interrupt Service Routine to be run with a timer. The timer calls the ISR every millisecond. Data is passed out of our ISR via queues to be printed in our main FSM. Our main FSM allows us to turn the ISR off after 1500s to not overload the queue as the ISR gets called much faster than the processor is able to print
